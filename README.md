# 船闸调度规则模型
>1.结合船闸交通运行规则，建立实用的内河**单级复线**船闸交通模型。  
2.考虑平衡船闸交通负荷与服务质量的时间约束，提出调度目标函数和约束条件，设计出船舶过闸交通模型的整体框架及*闸室分配、闸室编排、运行调度*三个耦合子模块的启发式求解算法。  
3.模型用京杭运河淮安船闸实际过闸数据进行验证，结果能够准确反映船闸实际运行状况，可用于分析不同条件下的船闸运行状态和最大通过能力。
